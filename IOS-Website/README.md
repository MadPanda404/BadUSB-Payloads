# IOS Website

<b>This script streamlines the process of opening a specific website on an iOS iPhone using the iOS Spotlight search function. Instead of manually navigating through Safari or searching for bookmarks, users can simply invoke Spotlight and enter a predefined keyword to access their desired website effortlessly.</b>

## Description

> [!WARNING]\
> <b>Please enjoy and use RESPONSIBLY‚ùó</b>

This payload open your desired website on iOS 17 and older versions! Save time with this convenient shortcut.

---------------------------------------------------------------------------------------------------------------------------------------------------------

### Dependencies

* An internet connection
* Any IOS Version

### Executing program

* Plug in your device / Connect through Bluetooth
* Website will be opened through spotlight
